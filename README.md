# Análisis y Clasificación de Datos con Machine Learning

## Descripción
Proyecto de análisis de datos utilizando modelos de machine learning para clasificación.

## Tecnologías
- Python
- Pandas
- NumPy
- Scikit-learn

## Modelos utilizados
- Regresión Logística
- Random Forest
- Gradient Boosting

## Métricas
- Accuracy
- Precision
- Recall
- F1-score

## Objetivo
Comparar distintos modelos de clasificación y evaluar su rendimiento.
